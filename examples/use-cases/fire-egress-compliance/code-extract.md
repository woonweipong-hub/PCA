# Building Code Extract (Sample)

Project: Riverside Mid-Rise Residential Block B

1. Minimum exit stair width for occupant load of this type is 1.5m.
2. Maximum travel distance to nearest protected exit stair is 30m.
3. Dead-end corridor length should not exceed 6m.
4. Refuge floor must be connected to both protected stair cores.
5. Smoke-stop lobby should be provided at each staircase entry.
